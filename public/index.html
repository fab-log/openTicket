Welcome to openTicket

openTicket helps you organize complex tasks or projects. It can also be used as a help desk manager with multiple people working on the same task.

Start by clicking the plus icon and add your first ticket. Each time you or one of your workmates comes back to this topic you can add an entry providing new information or a new status.

Styles

Pick from different colors to get an individual accent for each task

The default style displays a table offering sorting and filtering options.

You can opt for an alternative, more intuitive style showing the tickets as colored bubbles.

Dark or light

Switch to the light mode if you feel more comfortable with that.
