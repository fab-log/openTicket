Privacy

Very simple.

  • No advertising.
  • No trackers.
  • No preconnect. (download of third party content)

Data you provide will be stored in a database on the server. The connection to the server is secured using the HTTPS protocol. In its current beta state, the provided data is not encrypted on the server. Therefore, anyone with access to the server can read your data and your tasks (encryption will be available in the near future). No data is stored in another location, nor will any data be transferred to other parties.
To delete your data, you can use the corresponding built-in feature, which can be found in the settings menu of this application.
